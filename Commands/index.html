Oasis Commands

Commands that can be used anywhere

Command Name Command Variables Command Description Command Requirement
,goto place Moves you to another place More than 50 Calories
,eat amount of calories Moves calories from your food inventory to your calories More than amount of calories given in the variable
,ride place Moves you to another place, costs coins but less calories. More than 5 coins and 15 calories
,places none Sends a list of all the places that you can go to. None
,countries none Sends a list of all countries None
,give user(ping), amount of coins you want to give to the user Gives the given amount of coins to the given user More than the amount of coins you are giving
,commands None Sends almost all commands that you can currently use None
,stats None Sends the value of all your variables None
,info User(ping) Sends all values of variables given user has None
