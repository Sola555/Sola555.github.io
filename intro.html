How to play

Introduction on How To Play

Starting

Once you have added the bot to your server, either in the server or in the DMs with you and the bot, say ,start. This will set up all your data in our database which is actually just a simple text document that you can access with the ,txt command, but you shouldn't need to access this file.

Going to places

You can go to many places by using the ,goto {place} command. You can see all the places by using the ,places command. The places command will give you the short names of each place. When using the ,goto command, replace the {place} with the letter or set of letters that are given from the ,places command. When you go to a place, you can unlock certain commands. Go to the commands section to view usable commands for each place. It will also show you the goto command for that place.

Objective of the game

The objective of the game is to have a huge amount of many collectibles. You can see each collectible bellow. Click on the buttons inside the tabbed menu:

Coins

You can get coins by working. Coins are used to pay to buy various items and various shops. You can gamble coins at the Casino.

Health

There are two places where you can use commands to get healthier. One is the park, and the other is the sports and excersize center.

Calories

You can get calories by buying and eating food. Buy food at the supermarket, or automatically get food in your calories inventory from the cafe or restaurant. To eat food bought from the supermarket, you must use the ,eat command. See more information about this command in the commands section

Rich Stats

Rich stats are stats that you can achieve when you buy something at the Jewelry shop. Use the ,jewelry command to view what jewelry you can buy!

Happiness

The 2 main places you can achieve more happiness points are at the desires shop and the amazement park. At the desires shop, just say ,buy and anything you want, and you will get alot of happiness points. You can use various commands at the amazeent park to get happiness points, but keep in mind that you can also lose happiness points there!

Helpful commands that have a huge advantage

In the world of Oasis, there are some commands that you can use to get so many things super easily. For example, the ,daily command is a useful command that has no payment to use! The downside about this command is that you can only use it once per day. That means you have to wait 24 hours until you can use it again. This command is recomended for using because even though you can't use it as often, it gives you alot of things. There are commands like this that you can see in the commands section of this website. Another useful command or rather place is the bank. The bank is way better than the bank in real life because there is a 1% interest rate for everyone daily! This means you gain 1% of the current mass of your bank account everyday! This allows you gain coins even if you're busy and not on discord! You will learn how to use the bank in the next section.

The Bank

The bank ,as mentioned in the previous section, is a great place to gain coins. You can go there by using the ,goto b command! You can deposit your coins into your bank by using the ,deposit {amount} command. Replace amount with the number of coins you want to deposit.
